## Why Data Structures Matter

Data structures are the building blocks of every non-trivial program you'll write. Choosing the right one can be the difference between an O(n²) solution and an O(n log n) solution.

## Arrays

An array stores elements in contiguous memory and gives you **O(1)** random access.

```python
nums = [10, 20, 30, 40]
print(nums[2])  # 30
```

## Stacks

A stack is **LIFO** (last in, first out). Python lists work great as stacks.

```python
stack = []
stack.append("a")
stack.append("b")
stack.append("c")
print(stack.pop())  # c
```

## Queues

A queue is **FIFO** (first in, first out). Use `collections.deque` for O(1) popleft.

```python
from collections import deque

queue = deque()
queue.append("first")
queue.append("second")
print(queue.popleft())  # first
```
